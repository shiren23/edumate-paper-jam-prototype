📋 页面流程说明

  1. 打印中 → 打印机卡纸 → 系统报错并中断当前任务
  2. 卡纸提示页面 → 弹窗询问"是否打印后续试卷？"
  3. 若选"否" → 打印完成页面（结束状态） → 提示取出剩余试卷
  4. 若选"是" → 重匹配界面 → 勾选被卡纸试卷 → 点击"下一步"
  5. 续打确认页面 → 确认打印范围 → 点击"开始打印"
  6. 回到打印流程 → 完成后 → 打印完成页面（成功状态）
